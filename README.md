# Calculating Eigenvalues

Проект реализует численные методы для нахождения собственных значений и собственных векторов матриц на C++.

## Реализованные методы

### 1. Степенной метод (Power Method)
Находит **максимальное по модулю собственное значение** и соответствующий собственный вектор.

**Особенности:**
- Простая реализация
- Быстрая сходимость для доминирующего собственного значения

---

### 2. Обратный степенной метод (Inverse Power Method)
Позволяет найти собственное значение, ближайшее к заданному приближению `σ`.

**Особенности:**
- Использует решение СЛАУ (PLU-разложение)
- Подходит для поиска конкретных собственных значений

---

### 3. QR-алгоритм
Находит **все собственные значения матрицы**.

**Что реализовано:**
- Приведение к форме Хессенберга
- QR-разложение с помощью вращений Гивенса
- Сдвиг Уилкинсона для ускорения сходимости
- Дефляция (уменьшение размерности задачи)

---
